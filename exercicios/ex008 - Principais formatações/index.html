Principais formatações

Negrito / Destaque

Nessa Frase temos um termo em Negrito usando tag B(Não Semântica)

Nestra frase, tenho um termo em Destaque usando a tag STRONG(Semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I(Não semântica)

Nesta frase, temos um termo em ênfase usando a tag EM(Semântica).

Recomandação: Tanto negrito como itálico são funções para o CSS (Formas) e sentido são para o HTML

Observações: Selecionando a frase/palavra em seguida usando CTRL + SHIFT + P, pesquisa por "ABB" e clicando o "Emmet: Wrap with Abbreviation" com isso você pode utilizar qualquer tag como por exemplo "P" e "I" ou qualquer tag que for usar.

Texto Marcado

Podemos criar tambem um texto marcado usando a tag MARK.

E no outro parágrafo, temos outro texto marcado no final.

Observações: Para mudar a cor do texto marcado é utilizado "MARK style="background-color:..." e em seguida escolher a cor que você quer e também mudar ela só clicando na caixa de cor que está no código.(Nesse caso ele só tem influência onde você selecionou a palavra e não nas outras tags iguais em outros parágrafos em cor padrão que é amarelo).

Caso você queira mudar a cor para todas as tag de "MARK", basta ir em na chave "<head>" e dentro dela usar a tag "<style>" dentro da tag use o mark e depois "{}", dentro coloque a tag "background-color" e em seguida escolha a cor.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo (Tag big e small).

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.(tag del)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.(tag ins)

Existe também o sublinhado com a tag U(não semântica).

Texto sobrescrito

Para inserir coisas do tipo "x20+3" para "x20+3" (tag sup).

Texto subscrito

Para inserir coisas do tipo "H2O" para "H2O" (tag sub).
