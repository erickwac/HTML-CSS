Usando links externos

Você pode acessar o meu repositório público no GitHub

Você tambem pode acessar o meu canal gratuito no Youtube

Envelopando a frase/palavra com Tag "a", ele vai te pedir o link e vai transforma-lo em direcionamento para esse link sobrepondo no lugar do site em que você estava.

Dentro da Tag "a" existe o recurso da Tag "target="_blank" rel="external"", a tag "target="_blank"" é a indicação para abrir o link em outra aba e a tag "rel="external"" é a indicação para o mecanismo de busca que esse link é externo.

Usando links interno

Esta é a primeira pagina do site. Se você quiser, pode acessar também a minha segunda página

Você também pode acessar a nossa página de notícias

Na tag "a href="pag002.html" rel="next" target="_self"", a tag "_self" é a indicação que o link está dentro do seu site e a tag "rel="next"" é a indicação para o mecanismo de busca entender que é a proxima/seguinte pagína do seu site.
